## Getting Started

To get started, just keep on reading!

### Installation

1. Download the repo from Github and unzip the file.
2. Install dependencies
    ```
    npm install
    ```
3. Run the project locally
    ```sh
    npm start
    ```
4. Be a true developer
    ```js
    Google any errors you stubmle into
    ```
